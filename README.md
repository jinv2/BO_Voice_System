# 驳 · 万象智能体 (Borg Universal Agent Matrix)

> “中曲之山，有兽焉，其状如马而白身黑尾，一角，彘牙伸爪，其音如鼓，其名曰驳，是食虎豹，可以御兵。” —— 《山海经》

## 🐉 项目愿景
在 2026 年的跨平台办公丛林中，**【驳 · 万象智能体】** 是由 **神思庭 (Shensi-ST) / 天算AI实验室** 打造的首席操作系统操控 Agent。它不仅是一个简单的脚本工具，更是一个拥有“L2 决策大脑”的操作系统执掌者。

正如《山海经》中的“驳”能食虎豹、御兵灾，本智能体旨在接管复杂的键鼠操作、剪贴板监控、文件治理与系统自动化，将一切繁琐的重复性劳动转化为毫秒级的逻辑流。

## 🚀 核心特性
- **L2 万象归一逻辑**：内置原生逻辑引擎与本地 LLM (Ollama) 双重驱动，精准理解意图。
- **物理接管 (Physical Control)**：深度模拟键盘、鼠标、剪贴板，支持跨端（Windows/Mac/Ubuntu）执行。
- **商务项目治理**：自动化整理混乱目录，一键生成标准商务项目架构。
- **全自动跨平台构建**：集成 GitHub Actions，自动生产 Windows/Mac/Linux 三端可执行文件。
- **无感交付**：支持 Netlify 部署的专属 Landing Page。

## 🛠️ 快速开始
请阅读详细的 [安装教程 (INSTALLATION_GUIDE.md)](./INSTALLATION_GUIDE.md) 获取针对您系统的配置说明。

1. **环境准备**：
   - Python 3.10+
   - Ollama (可选，用于增强语义理解)
2. **启动程序**：
   ```bash
   python src/bo_ui.py
   ```
3. **打包交付**：
   ```bash
   python build_cross_platform.py
   ```

## 🌐 部署说明
本项目原生支持：
- **GitHub Actions**：代码推送至标签（如 `v1.0`）时，自动在 Releases 产生三端成品。
- **Netlify**：根目录下的 `index.html` 可一键部署为产品展示网关。
