# 神思矩阵 · 驳 (BO) 全功能使用说明书 (2026 旗舰版)

欢迎使用 **“驳” (BO)** — 您的通用计算机控制智能体。本手册详尽列出了 “驳” 的核心功能矩阵及其使用方法，助您从初学者成长为系统指挥官。

---

## 🚀 核心理念：L2 级决策大脑
“驳” 不仅仅是一个工具，它是您的 **数字门客**。它具备自主意图识别与路径规划能力。您只需下达模糊目标，由 “驳” 完成逻辑拆解与物理执行。

---

## 🛠️ 功能矩阵 (48+ 技能深度集成)

### 1. ⚡ 高频物理操控 (High-Frequency Matrix)
直接模拟系统底层输入，接管繁琐的键鼠操作。
- **复制/粘贴**：`“帮我复制这段话”` / `“粘贴到这里”`
- **文件重命名**：`“给选中的文件改名”`
- **保存/刷新**：`“保存当前进度”` / `“刷新浏览器页面”`
- **窗口管理**：`“关闭当前标签页”` / `“模拟回车键”`
- **多媒体控制**：`“音量调高/调低”` / `“静音”`
- **系统锁定**：`“我要离开了，锁屏”` (全平台适配)

### 2. 📁 商务项目治理 (Protocol · 乾坤)
一键构建标准化办公环境，对抗目录熵增。
- **一键开案**：输入 `“为 [客户名] 建立项目架构”`。
  - *执行结果*：自动生成包含 合同、财务、会议纪要、素材、交付物 的标准化目录树。
- **逻辑归档**：输入 `“整理当前目录”`。
  - *执行结果*：Agent 将自动分类 `.py` 到 `src`，`.md` 到 `docs`，`.json` 到 `config`。
- **新建文档/目录**：`“建一个新文件夹叫 [名称]”` / `“写个备忘录项目进度”`。

### 3. 🔍 数据情报挖掘 (Data Intelligence)
深层扫描本地文件，提取关键商务信息。
- **信息提取**：
  - `“提取桌面上所有的联系邮箱”`
  - `“查找这个目录下的所有电话号码”`
  - `“统计本月所有涉及金额的记录”`
- **文件分析**：`“看看今天桌面新加了哪些文件？”`

### 4. ✍️ 生成式文案创作 (Generative Creative)
集成 LLM 能力，为您处理专业的文字工作。
- **商务起草**：`“写一封给张总的正式邀请函”`
- **智理建议**：`“根据今天的日程推荐午饭”`
- **内容合成**：`“帮我总结这个项目的核心逻辑”`

### 5. 💻 通用系统控制 (Universal Shell)
无论 Windows PowerShell 还是 macOS Zsh，指令自动转义。
- **进程管理**：`“查看内存占用最高的三个进程”`
- **网络诊断**：`“测试网络延迟”`
- **文件搜索**：`“寻找最近三天修改过的包含 'contract' 的文件”`
- **应用启动**：`“打开终端”` / `“运行浏览器”` / `“启动计算器”`

## 💡 诊断模式：自我修复与引导
如果您要求执行一项当前环境不支持的任务（例如 `“剪辑视频”`），“驳” 会自动触发 **【能力诊断报告】**。它会解释原因，并为您提供一键安装所需组件（如 `ffmpeg`）的指令。
